# Fullstack JavaScript Application - Frontend

This is the frontend part of the Fullstack JavaScript application built with React. 

## Getting Started

To get started with the frontend, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd fullstack-js-app/frontend
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the application**:
   ```bash
   npm start
   ```

## Folder Structure

- `src/`: Contains all the source code for the frontend application.
  - `components/`: Reusable components used throughout the application.
  - `pages/`: Components that represent different pages in the application.
  - `utils/`: Utility functions for making API calls to the backend.
  - `App.js`: Main component that sets up routing and renders other components.
  - `index.js`: Entry point for the React application.
