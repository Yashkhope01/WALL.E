# Fullstack JavaScript Application - Backend

This is the backend part of the Fullstack JavaScript application. It is built using Node.js and Express.

## Setup Instructions

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd fullstack-js-app/backend
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the application:**
   ```
   npm start
   ```
