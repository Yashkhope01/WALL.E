# Project Overview

This is a full-stack JavaScript application that consists of a backend built with Node.js and Express, and a frontend built with React. 

## Project Structure

The project is organized into two main directories: `backend` and `frontend`.

### Backend

- **Directory**: `backend`
- **Entry Point**: `src/app.js`
- **Controllers**: `src/controllers/index.js`
- **Routes**: `src/routes/index.js`
- **Models**: `src/models/index.js`
- **Middleware**: `src/middleware/auth.js`
- **Configuration**: `package.json`

### Frontend

- **Directory**: `frontend`
- **Entry Point**: `src/index.js`
- **Main Component**: `src/App.js`
- **Reusable Components**: `src/components/index.js`
- **Pages**: `src/pages/index.js`
- **API Utilities**: `src/utils/api.js`
- **Configuration**: `package.json`

## Setup Instructions

### Backend

1. Navigate to the `backend` directory.
2. Install dependencies using `npm install`.
3. Start the server with `npm start`.

### Frontend

1. Navigate to the `frontend` directory.
2. Install dependencies using `npm install`.
3. Start the React application with `npm start`.
